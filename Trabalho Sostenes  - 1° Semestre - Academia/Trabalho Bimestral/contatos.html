Índice de Massa Corporal (IMC)
• Para o que serve?
Essa técnica é utilizada para verificar o estado nutricional e observar se o indivíduo está dentro dos padrões de normalidade com relação ao seu peso e estatura.
• Porque devo fazer?
Este cálculo é uma forma simples e de grande importância para detectar se você apresenta um grau de desnutrição, se está no padrão de normalidade, sobrepeso, obesidade ou obesidade mórbida.
• Avisos importantes:
Deve-se considerar que apenas este cálculo não é suficiente para avaliar corretamente o estado nutricional. É necessário avaliar a massa muscular, hidratação, atividade física, etc.
